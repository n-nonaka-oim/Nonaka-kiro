### m_smtp_config（接続プロファイルマスタ・複数行運用）【db_common_dev】

複数行の接続プロファイルを保持する。各行が1つのSMTP接続先設定を表す。`config_key` を主キーとし、送信ジョブ(`t_smtp_queue`)が `config_key` で参照する。送信元アドレス・送信元名・テスト送信先・PDF保管先ディレクトリは保持しない（差出人はジョブが保持、PDFはジョブの `pdf_path` がフルパス保持）。

**送信モードは `fax_domain` の値の形で決まる**（SmtpAgent `ResolveToAddress` が判別）:
- **空** = メール直送モード（宛先はメール形式＝@必須。@なしはエラー）
- **`@` 始まりのドメインのみ**（例 `@faxmail.com`）= FAX送信モード（宛先はFAX番号＝@なし・数字。@混入や数字なしはエラー。数字正規化 先頭0→81＋ドメイン付与）
- **完全アドレス**（`@` の前に局所部あり、例 `0064871033@faxmail.com`）= 固定宛先モード（宛先を無視し `fax_domain` の値へ送信＝テスト用）

運用する `config_key` は **`mail`／`fax`／`test-fax`** の3種。旧 `Material`／`test` は廃止（2026/07/03・投入側 config_key 選定を fax/test-fax 化）。

| 列名 | 日本語名 | 型 | 必須 | 備考 |
|------|----------|-----|------|------|
| config_key | 接続プロファイルキー | nvarchar(40) | ○ | PK（`mail`/`fax`/`test-fax`） |
| host | SMTPホスト | nvarchar(100) | ○ | 既定 172.16.128.81 |
| port | ポート | int | ○ | 既定 25 |
| fax_domain | FAXゲートウェイドメイン | nvarchar(100) | | 値の形で送信モードを判別（空=メール直送 / @始まりドメイン=FAX送信 / 完全アドレス=固定宛先） |

| config_key | host | port | fax_domain | 送信モード・用途 |
|------------|------|------|------------|------|
| mail | 172.16.128.81 | 25 | （空） | メール直送（宛先は@必須） |
| fax | 172.16.128.81 | 25 | @faxmail.com | FAX送信（本番・番号正規化＋ドメイン付与） |
