# session-memo（セッション進捗ログ・ワークスペース共通）

本フォルダはワークスペース横断の**セッション進捗ログ**を集約する。steering / specs と同じ `.kiro/` 配下に置き、セッション再開時に確実に参照できるようにする。

- ファイル名: `session-memo-YYYYMMDD.md`
- 旧配置 `MaterialModule/Doc/` から移設（2026/07/01）。以降はここが正。
- 各プロジェクトモジュール固有のドキュメントは各モジュールの `Doc/` で管理する（進捗ログのみ本フォルダに集約）。

## 運用
- セッション開始時: 最新の `session-memo-YYYYMMDD.md` を読む。
- セッション終了時: 当日分を作成・更新（完了作業・未完了タスク・参照ファイル）。
- 再開合図: 「再開します、session-memoを確認」。
